# MODULOS 3 Y 4 CISCO

## MODULO 3 

- Redes inalámbricas

  Los teléfonos móviles utilizan ondas de radio para comunicarse con antenas y así poder realizar llamadas y enviar mensajes. La señal viaja entre torres hasta
  llegar al destino. Existen diferentes generaciones de redes móviles como 3G, 4G y 5G, siendo 4G la más utilizada actualmente.

  Además, los dispositivos pueden conectarse mediante Wi-Fi para acceder a redes locales e Internet. También existen tecnologías como Bluetooth, que permite la
  comunicación a corta distancia entre dispositivos, y NFC, que permite intercambiar datos cuando los equipos están muy cerca.

- Conectividad de dispositivos móviles

  Los dispositivos móviles pueden conectarse a redes Wi-Fi, pero es importante tomar medidas de seguridad como no enviar datos sin cifrar, usar VPN, activar la
  seguridad del router y utilizar WPA2 o superior.

  Los sistemas operativos más comunes son Android e iOS, y los dispositivos se conectan automáticamente a Wi-Fi si está disponible; de lo contrario, usan datos
  móviles. El Bluetooth permite conectar accesorios de forma sencilla y con bajo consumo de energía. Para emparejar dispositivos, ambos deben estar visibles y puede
  requerirse un PIN. Durante este proceso, los dispositivos comparten información como nombre, servicios y características.

## MODULO 4

- Conceptos Básicos de Redes Domesticas

  Las redes domésticas funcionan como una pequeña LAN que conecta la red pública del proveedor de servicios con los dispositivos del hogar. El router es el núcleo
  de esta estructura, actuando como un equipo integrado que permite la comunicación tanto por cable como de forma inalámbrica. Gracias este centro de conexión, una
  gran variedad de dispositivos como consolas, televisores inteligentes, cámaras de seguridad y sistemas de control de temperatura pueden intercambiar información
  entre sí y acceder a internet de manera sencilla y económica.

- Tecnologías de Red en el Hogar

  La conectividad en el hogar se divide principalmente en tecnologías inalámbricas y cableadas. Las inalámbricas utilizan ondas electromagnéticas, específicamente
  en las bandas de 2.4 GHz y 5 GHz, bajo estándares que ofrecen mayor alcance y potencia que el Bluetooth.

  Por otro lado, la conexión por cable sigue siendo fundamental para obtener estabilidad, utilizando mayormente el protocolo Ethernet con cables de par trenzado
  que ayudan a reducir las interferencias eléctricas en la transmisión de datos.

- Estándares Inalámbricos

  El entorno Wi-Fi se rige por el estándar IEEE 802.11 y requiere la configuración de parámetros específicos para funcionar correctamente. Entre ellos destaca el
  SSID, que es el nombre que identifica a la red para que los dispositivos puedan unirse a ella, y el "Modo de Red", que determina la compatibilidad entre
  versiones antiguas y modernas de la tecnología. 

  También es posible gestionar la transmisión del SSID para que sea visible o permanezca oculto, y elegir canales de transmisión automáticos o manuales para
  optimizar el rendimiento y evitar interferencias con otras redes cercanas.

- Configurar un enrutador domestico

  Para configurar un router doméstico, lo ideal es conectar una computadora a un puerto LAN mediante un cable Ethernet. El router, gracias a su servidor DHCP, le
  asignará automáticamente una dirección IP a la computadora para que puedan comunicarse. Antes de entrar a la interfaz de configuración en el navegador, es
  fundamental elegir un nombre de red que no revele la marca o el modelo del dispositivo, ya que esa información facilitaría a posibles atacantes encontrar
  vulnerabilidades específicas en internet.

  En cuanto a la seguridad y el control de acceso, se pueden usar herramientas como el filtrado de direcciones MAC, que permite identificar y autorizar solo a
  dispositivos específicos en la red inalámbrica. Además, es común configurar un acceso para invitados; esto crea un SSID independiente que ofrece conexión a
  internet de forma abierta, pero mantiene la red principal privada y protegida de ojos ajenos.
